# Problem 04: Even or Odd Number Check

## Objective
Write a Java program to check whether a given number is **even or odd**.

## Requirements
- Declare an integer variable `number`
- Use conditional statements (`if-else`)
- Print:
  - "Even Number" if the number is even
  - "Odd Number" if the number is odd

## Constraints
- Do not use loops
- Use modulo (%) operator
- Keep the code simple and readable

## Expected Output Examples
Input: 10  
Output: Even Number  

Input: 7  
Output: Odd Number
