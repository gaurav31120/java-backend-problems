## Problem Statement
Write a Java program to swap two numbers without using a third (temporary) variable.

## Requirements
1. Declare two integer variables `a` and `b`
2. Print values before swapping
3. Swap the values WITHOUT using any third variable
4. Print values after swapping

## Constraints
- Use only arithmetic operators (+, -)
- Do not use Scanner
- Do not use any extra variable
- Use int data type only

## Expected Output (Example)
Before Swap: a = 10, b = 20  
After Swap: a = 20, b = 10
