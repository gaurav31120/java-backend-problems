Q1. What is the difference between JDK and JRE?

Answer:
JDK is used for development and includes compiler tools, while JRE is used only to run Java applications.

Q2. What is the role of JVM?

Answer:
JVM executes Java bytecode and provides platform independence.

Q3. What is bytecode?

Answer:
Bytecode is an intermediate code generated by the Java compiler that runs on JVM.

Q4. Is JVM platform dependent?

Answer:
Yes. JVM is platform dependent, but Java programs are platform independent.
